Online Vehicle Rental System

Online Vehicle Rental System is a web-based application that allows users to rent cars, bikes, and other vehicles online. Users can register, log in, view available vehicles, check prices, and book vehicles easily.

Features

- User Registration and Login
- View Available Vehicles
- Book Vehicles Online
- Check Vehicle Details and Prices
- Manage Customer and Booking Records
- Payment Management

Frontend

- HTML
- CSS
- JavaScript

Backend

- PHP

Database

- MySQL

How to Run

1. Download the project
2. Open the project folder in VS Code or XAMPP htdocs folder
3. Start Apache and MySQL in XAMPP
4. Import the database in phpMyAdmin
5. Run the project in browser

Purpose

This project helps reduce paperwork, saves time, and provides an easy way to manage vehicle rentals online.
